## FAQ

### How does billing work?
```
Monthly plans renew on the date you signed up. Annual plans save 20% 
and renew yearly. You can upgrade/downgrade anytime.
```

### What happens if I exceed my limit?
```
Starter: API calls stop until next month
Developer/Professional: Auto-charge for extra requests
Enterprise: Unlimited, no overages
```

### Can I change plans?
```
Yes! Upgrade anytime and pay the difference. Downgrade at renewal.
Pro-rated credits for annual plans.
```

### Is there a free trial?
```
Starter is free forever. Developer & Professional include 14-day free 
trials. Enterprise includes 30-day POC period.
```

### What payment methods do you accept?
```
Credit card, bank transfer (annual), crypto (USDC, SOL)
Enterprise: Custom invoicing available
```

### Do you offer refunds?
```
30-day money-back guarantee on annual plans.
Monthly plans are non-refundable but you can cancel anytime.
```

### What's included in support?
```
Starter: Community forum
Developer: Email support (24h response)
Professional: Email + Slack (4h response)
Enterprise: 24/7 phone + dedicated account manager
```

### Can I host it myself?
```
Enterprise plan includes on-premise deployment option.
Self-hosted requires annual contract.
```
